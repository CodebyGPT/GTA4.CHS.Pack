# 适用于GTA4三合一版本(1.2.0.43及以上)的汉化+增强资源，确保游戏能在现代环境下正常运行的同时获得更完善的游玩体验。

## 适配的游戏
Grand Theft Auto IV 1.2.0.43+（即 Grand Theft Auto IV: The Complete Edition），目前仅在[Steam](https://store.steampowered.com/app/12210/)最新版本上测试通过，其他版本自行测试。

## 获取方式和使用方法
[本仓库Release页面](https://github.com/CodebyGPT/GTA4.CHS.Pack/releases)。

## GTA4开启HDR的方法
GTA4是D3D9游戏，因此不兼容Windows11的自动HDR功能（Windows Auto HDR目前仅D3D11+游戏）。

**英伟达NVIDIA显卡用户**：先在[NVIDIA App](https://www.nvidia.cn/software/nvidia-app/)中为GTA4启用RTX HDR功能，并在GTA4游戏设置中关闭窗口化和无边框窗口化选项（~\plugins\GTAIV.EFLC.FusionFix.cfg中应为“Windowed = 0”）使游戏可以独占全屏显示。

**其他显卡用户**：（暂无）
