# 适用于 GTA 4 三合一版本（1.2.0.43及以上）的汉化+增强资源集合，确保游戏能在现代环境下正常运行的同时获得更完善的游玩体验。

## 适配的游戏
Grand Theft Auto IV 1.2.0.43+（即 Grand Theft Auto IV: The Complete Edition），目前仅在最新 [Steam](https://store.steampowered.com/app/12210/) 版本上测试通过，其他版本自行测试。

## GTA4开启HDR的方法

*GTA4 是 D3D 9 游戏，因此不兼容 Windows 11 的自动 HDR 功能（Windows Auto HDR 仅支持 DirectX 11 或 DirectX 12 游戏）。*

**英伟达 NVIDIA 显卡用户**：先在 [NVIDIA App](https://www.nvidia.cn/software/nvidia-app/) 中为 GTA 4 启用 RTX HDR 功能，并在 GTA 4 游戏设置中关闭窗口化和无边框窗口化选项（~\plugins\GTAIV.EFLC.FusionFix.cfg中应为“Windowed = 0”）使游戏可以独占全屏显示。

**其他显卡用户**：（暂无）
